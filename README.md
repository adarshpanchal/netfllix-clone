# 🎬 Netflix Clone (Frontend)

A fully responsive **Netflix Clone Webpage** built using modern frontend technologies. This project replicates the UI/UX of Netflix, including homepage layout, banners, movie sections, and interactive elements.

---

## 🚀 Features

* 🔥 Modern Netflix-style UI
* 📱 Fully responsive (Mobile + Desktop)
* 🎥 Movie/Show banners with hover effects
* 🎨 Gradient overlays & stylish cards
* 📂 Organized sections (Trending, Popular, Originals, etc.)
* ⚡ Fast and lightweight frontend

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3** (Flexbox, Grid, Gradients, Clip-path)
* **JavaScript (Vanilla JS)**

## 🎯 How to Run

1. Clone the repository:

   ```
   git clone https://github.com/your-username/netflix-clone.git
   ```

2. Open the project folder:

   ```
   cd netflix-clone
   ```

3. Run the project:

   * Simply open `index.html` in your browser
